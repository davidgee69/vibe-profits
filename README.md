# Vibe Profits

Landing page for Vibe Profits — the education project teaching agencies, consultants, and creators to ship revenue-generating tools in days, without developers.

**Live:** vibeprofits.com
**By:** David Gonzalez

## Stack

Static HTML + CSS. No build step. Deploys directly to Vercel from this repo.

## Files

- `index.html` — landing page with email capture
- `thanks.html` — post-submit confirmation
- `terms.html` — Terms & Conditions
- `privacy.html` — Privacy Policy
- `404.html` — not-found page
- `styles.css` — shared stylesheet
- `logo.svg` — logomark
- `favicon.svg` — browser tab icon
- `images/` — event press assets (Connect 2026)

## Local preview

```
open index.html
```

## Deploy

Push to `main`. Vercel auto-deploys.
